# Biography

## Description
This program prints a short biography using simple text variables (`string`) and numbers (`int`).

## How to run
- 1 Open the terminal in VS Code.
- 2 Compile the program:  
g++Biography.cpp -o Biography
- 3 Execute

.\ Biography.exe

## What I learned
- Use `string` variables to store text.
- Combine text and numbers in `cout`.
- Add line breaks using `\n`.
- Write multi-line console output.
