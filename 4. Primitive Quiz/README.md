# Primitive Quiz

## Description
A simple mulitple-choice quiz about primitive types in C++.
Its asks three questions, reads the user's answer (a/b/c), accepts uppercase or lowercase, count correct answers and prints the final score.

## How to run
- 1 Open the terminal in VS Code.
- 2 Compile the program:  
g++PrimitiveQuiz.cpp -o PrimitiveQuiz
- 3 Execute

.\ PrimitiveQuiz.exe

## What I learned
- Use `char` to store single letter.answer.
- Read user input with `cin`.
- Compare answers with `if` statements.
- Normalize input using `tolower` to accept A/a.
