# Hello

## Description
This program asks the user for their full name and displays a personalized greeting message.  
It demontrates how to read text input and print output using `cin`, `getline`, and `cout`.

## How to run
- 1 Open the terminal in VS Code.
- 2 Compile the program:  
g++Hello.cpp -o Hello
- 3 Execute

.\ Hello.exe

## What I learned
- Use `string` variables to store text.
- Use `getline(cin, variable)` to read full sentences with spaces.
- Use `cout` to display messages.
- Combine text and variables in output using `<<`.
