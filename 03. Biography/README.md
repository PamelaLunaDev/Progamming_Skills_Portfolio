# Biography

## Description
This program prints **name**, **hometown** and **age** on separate lines using only one `cout`.  
Each piece of information is stored in a variable with an appropriate data type.

## How to run
- 1 Open the terminal in VS Code.
- 2 Compile the program: 
g++ Biography.cpp -o Biography

- 3 Execute
- On Windows:  
.\Biography.exe

- On macOS / Linux:  
./Biography

## What I learned
- How to create and store information in variables of different data types `string` and `int`.
- How to print multiple values using **a single `cout` statement**
- How newline characters `\n` control the formatting of output.
- The basic structure of a simple C++ program.
